# ios_mvp

This folder is used for iOS MVP support files.

It is still active, but mainly for:

- model export scripts
- generated ExecuTorch artifacts
- setup notes
- progress / status documents
- template source used as reference

Use this folder for:

- `export_yolo11_executorch.py`
- `build/detector.pte`
- `CURRENT_STATUS.md`
- `XCODE_SETUP.md`

Do not treat this folder as the primary running Xcode app.

The canonical app project is:

- `../RealtimeDetectionMVP/RealtimeDetectionMVP.xcodeproj`

The canonical app source folder is:

- `../RealtimeDetectionMVP/RealtimeDetectionMVP/`
