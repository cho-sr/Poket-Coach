# RealtimeDetectionMVP Template

This folder is a template / mirror copy of the MVP source.

It exists so that:

- MVP source can stay alongside export scripts and iOS notes
- reference code can be kept under `ios_mvp/`

Important:

- This is not the main Xcode app source currently used for device builds.
- Editing files here will not automatically change the running iPhone app.

Use the canonical app source instead:

- `../../RealtimeDetectionMVP/RealtimeDetectionMVP/`

Use the canonical Xcode project instead:

- `../../RealtimeDetectionMVP/RealtimeDetectionMVP.xcodeproj`
